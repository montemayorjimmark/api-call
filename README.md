## Project Title

Laravel API Call

## Project Description

This project is a simple CRUD API system developed using Laravel. The system allows users to perform Create, Read, Update, and Delete (CRUD) operations through API calls tested in Postman. The project uses MySQL database connection through XAMPP and Laravel Artisan commands such as `php artisan serve` to run the project locally. Different HTTP methods including GET, POST, PUT, PATCH and DELETE were used to demonstrate the functionality of the API.

## Setup Instructions

First, open XAMPP and start Apache and MySQL. After that, open phpMyAdmin and create the database for the project. Configure the database connection in the `.env` file, then open the terminal and run `php artisan migrate` to create the tables in the database. After the migration is complete, run `php artisan serve` to start the project on a local server. Finally, use Postman to test the CRUD API calls using GET, POST, PUT, PATCH and DELETE requests.
